列表

无序列表使用星号、加号或是减号作为列表标记：

  • Green
  • Blue

等同于

  • Red
  • Green
  • Blue

和

  • Red
  • Green
  • Blue
1

3

- Red
- Green
- Blue

有序列表则使用数字接着一个英文句点：

  1. Bird
  2. McHale
  3. Parish

数字并不会影响输出的 HTML 结果，也就是说上面的例子等同于：

  1. Bird
  2. McHale
  3. Parish
