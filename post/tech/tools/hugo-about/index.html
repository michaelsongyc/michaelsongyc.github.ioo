Hugo如何添加about页

2021-08-14
hugo
约 212 字 预计阅读 1 分钟

文章目录

  • 创建页面
  • 修改配置
  • 启动服务

创建页面

此处以even模板为例

1

hugo new post/about.md

文件创建完成后，会生成在 ${根目录}/content/post/目录下； 将about.md上移一层，到${根目录}/content/下, 和post目录同级。

修改配置

修改根目录文件config.toml，新增如下配置

1
2
3
4
5

[[menu.main]]
  name = "关于"
  identifier = "about"
  url = "/about/"
  weight = 50

weight 根据menu的顺序自定义数字即可；

启动服务

1

hugo server -D

访问：http://localhost:1313 即可查看效果；
